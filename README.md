# FullStack Internship Project

A Django-based To-Do List web application built as part of a Full Stack Internship project.

##  Features

- User Registration
- User Login & Logout
- Add, Edit, Delete Tasks
- Clean UI Design
- Django Backend Integration

##  Tech Stack

- Python(Django)
- HTML
- CSS
- Javascript
- SQLite

## How to run
- Activate the virtual environment: env/Scripts/activate
- Install the requirements: pip install -r requirements.txt
- Apply Database Migrations: python manage.py runserver
- Start the server :python manage.py runserver
